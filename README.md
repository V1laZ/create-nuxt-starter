# create-nuxt-starter

A CLI tool to quickly scaffold Nuxt.js projects with a pre-configured [starter template](https://github.com/V1laZ/nuxt-project-starter).

## Prerequisites
[pnpm](https://pnpm.io/)

## Usage
```bash
npx create-nuxt-starter <project-name>
```
